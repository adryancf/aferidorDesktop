1- TER O INNO SETUP INSTALADO
2- ABRIR O ARQUIVO InnoSetup_instalador
3- CERTIFICAR DE QUE AS PASTAS INSERIDAS NO INSTALADOR ESTÃO NO CAMINHO CORRETO ([Files])
4- COM TUDO CONFERIDO, APERTE F9 OU RUN LOCALIZADO NO TOPO DA TELA
5- AGUARDE ATÉ A FINALIZAÇÃO DA COMPILAÇÃO
6- NO FINAL ELE VAI INICIAR O INSTALADOR AUTOMATICAMENTE, FECHE-O
7- LOCALIZE O NOVO ARQUIVO DO INSTALADOR (PADRAO: C:\Users\0394\Desktop\Sistema_Auditoria\aferidorDesktop\Modulo\INSTALAÇÃO\InnoSetup)
8- COPIE O NOVO ARQUIVO DO INSTALADOR PARA A PASTA INTRANET:\sistemas\paginas\programas\aferidorDesktop\download
9- PRONTO, O NOVO INSTALADOR JA VAI ESTAR DISPONÍVEL PARA DOWNLOAD NO SITE DO AFERIDOR DESKTOP.

PS: Sempre que fizer uma atualização para o Aferidor Desktop, os usuários podem simplesmente executar o novo instalador. Não é necessário excluir a versão anterior do programa. O instalador substitui automaticamente os arquivos antigos pelos novos, garantindo que o programa seja atualizado sem a necessidade de intervenção manual.

Mas para que isso ocorra, crie um novo EXE com o aferidorDesktop modificado através do auto-py-to-exe.exe seguindo o tutorial localizado na raiz dos arquivos.
